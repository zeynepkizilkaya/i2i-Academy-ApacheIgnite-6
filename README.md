# Apache Ignite Homework

This project demonstrates the usage of Apache Ignite 3 Thin Client with Java.

## Features

- Apache Ignite 3 Thin Client
- Docker-based local deployment
- SQL table creation
- Subscriber data model
- Random usage simulation
- SQL INSERT, SELECT, UPDATE, and DELETE operations

## Technologies

- Java
- Apache Ignite 3
- Docker
- Maven
- Spring Boot

## Project Workflow

1. Connect to the local Apache Ignite node.
2. Create the `Subscriber` table if it does not exist.
3. Clear the table before each execution.
4. Insert five dummy subscribers.
5. Retrieve subscriber records.
6. Simulate random usage values.
7. Update subscriber records.
8. Print the final state of all subscribers.
